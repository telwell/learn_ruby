The Basics of HTML and CSS

In this lesson, you'll learn about the HTML markup language and the CSS styling language. While you're learning the basics, you'll probably alternate between awesome moments of "Holy cow, so that's how that works!" and the frustration of "Geez, there are so many things to remember..."

Hang onto those first moments and remember that you won't be able to memorize every element or selector on the first pass through the reading. Experienced developers still have to look things up on Google all the time! It can be helpful while you're learning to make a cheat sheet or take notes, but that's up to you.

You'll probably find that HTML will come easily and make intuitive sense (even if it's a bit awkward at first) and that CSS takes a bit more getting used to. CSS has more to learn since you've got to develop a mental model for how elements get targeted as well as remembering which styles are available for you to use. Even so, you can still pick up most of the basics you need in just a day or two of reading.

How This Lesson Works

In this lesson, you'll read up on HTML and CSS. The lessons after this one will give you a chance to see some examples of this stuff in action so you can get a sense for what's actually used versus what is "save for later" knowledge. At the end of this section, you'll be buiding websites of your own.

HTML and CSS are topics that have been covered at length by other people across the web and some of their resources are actually pretty darn good. For this lesson, we'll rely on Shaye Howe's excellent materials to cover most of the content. If that's not quite enough to get you comfortable, we've also provided tons of additional resources both below and in the Resources tab.

In particular, if you find yourself unsure of something, our backup resource is HTML Dog. It's a site by Patrick Griffiths, Tom Ashworth and Paul Cuthbertson which contains all kinds of yummy HTML and CSS (and Javascript) lessons written in Plain English.

Now that we've got that out of the way, let's get started.

Additional Help if You Need It

If you aren't sure about something in HTML, Read through the HTML Dog Beginner HTML Guide.

If you aren't sure about something in CSS, Read through the HTML Dog Beginner CSS Guide or their Intermediate CSS Guide.

If you just can't learn this stuff without applying it along the way, you could instead try the Codecademy HTML/CSS track, though you'll need to go back and read through Shaye Howe's stuff to fill in the gaps.
