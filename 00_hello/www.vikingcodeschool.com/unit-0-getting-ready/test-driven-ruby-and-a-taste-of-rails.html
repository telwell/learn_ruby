Project: Ruby TDD and a Taste of Rails

This project pulls together the final set of tasks in order to make yourself ready for the program. It will dive deep into Ruby and give you a taste of Rails as well.

Required: Test First Ruby

Estimated Time: 5-10 hrs

You will need to be really comfortable with Ruby before we start. These resources will help you hammer down your understanding before you get in the door.

  1. Go through the first level of Code School's RSpec Course (the free one), which will get you exposed to RSpec, Ruby's testing language. It moves pretty fast so you certainly won't absorb everything, but you'll need a bit of familiarity with it for this next part...
  2. Do the Test First Ruby exercises that are located in this Github Repo (You can skip #13: XML). They are from testfirst.org/learn_ruby). Follow the instructions in the README. Basically, you'll need to fork our repo, solve the exercises, and submit your solutions as a pull request to us. This is exactly how you'll submit solutions during the course.

Strongly Encouraged: More Ruby plus Rails

Estimated Time: 15-20 hrs

The more you come in with, the more you'll get out of the program. If you've got time before you start, dive deeper into some of the course topics ahead of time by completing these resources:

  1. Read Peter Cooper's Beginning Ruby chapters 1-8 except 5. It's a bit old but the core concepts still hold strong.
  2. Do the Jumpstart Blogger Rails project. It will throw you in over your head and that's intentional (so don't be frustrated when it's difficult).
  3. Set up a blog. If you don't have one, use Tumblr because it's as easy as you can get. We'll discuss this a bit more during the first unit but it's very helpful to blog at the end of each week as a way to get yourself out there and review what you've learned. We feature student blogs on our social media accounts and on the website, so you'll get some free exposure there.

Optional but Encouraged: The Hartl Tutorial (~50 hrs)

Estimated Time: 50 hrs

  1. Do the Ruby on Rails Tutorial by Michael Hartl. It will take ~50 hours to complete so don't be alarmed if it's a long haul.

...Finished? Then you're all set and good to go!
